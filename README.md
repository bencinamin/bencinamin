## Featured project

### IMS Bearing Analysis

Exploratory analysis of vibration signals from the NASA IMS Bearing dataset to identify degradation trends in rolling bearings.

The project focuses on:

- Loading and organizing raw vibration signal files.
- Extracting simple time-domain indicators such as RMS, standard deviation, maximum absolute value and kurtosis.
- Comparing early, intermediate and late-stage signals.
- Visualizing degradation patterns over time.
- Translating technical results into maintenance-oriented conclusions.
